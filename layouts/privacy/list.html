Policy

Welcome to https://hutasker.netlify.app Throughout this Privacy Policy, "personal information" or "personal data" refers to any information that is unique to an individual, such as name, address, email address, phone number, IP address and other information that can reasonably identify an individual.

What Information Do We Collect

Information Collected Directly

You can generally visit our Site without having to submit any personal information. If you request more information, or input data into our AirTable sheet, we will collect information as follows. User Data: Anything input in the AirTable sheet will save to a version history. Participating in and submitting user data is optional and not required to use this theme and site.

Information Collected Indirectly

When you use or interact with the Site, we collect information about your use of the Site via a third party - Simpleanalytics. Please see below an embed of the traffic, open to the public.

Cookies

We don't set cookies, however the AirTable embed on the home page may interact with AirTable.

How we use Information

Being a template for a Static Site Generator, we aim to collect as little information as possible. However having a link to our AirTable sheet accessible to the public, any information input will be displayed for the purposes of providing this example site. We may embed the AirTable sheet into multiple sources like: this site, an embed or any other use.

Do Not Track

Our analytics has been set up to gather and measure the popularity of the theme, and as such doesn't respond to Do Not Track ("DNT") browser signals. For more information on DNT settings generally, please visit https://allaboutdnt.com. If you would like to block analytics, the script from simpleanalytics is generally in most block lists.

How Long Do We Keep Your Personal information?

The lifespan of the AirTable account and the use of the data will be retained for as long as needed to provide this site.

Data Security

Reasonable steps have been taken to protect the data and its security. We have put in place reasonably appropriate security measures designed to prevent the AirTable data from being accidently lost, used or accessed in an unauthorised way, altered or deleted. We limit access to the API keys and logins to protect the data template.
